# SOC Analyst Splunk Lab

This repository contains hands-on SOC investigation and SIEM analysis projects built using Splunk Enterprise.

The projects focus on:
- Log analysis
- Threat hunting
- Dashboard creation
- Suspicious activity investigation
- Threat intelligence enrichment
- Basic SOC investigation workflows

## Projects

### DNS Log Analysis with Splunk

A DNS-focused investigation project using over 422,000 DNS log events.

Main topics covered:
- DNS traffic analysis
- Rare DNS query hunting
- Suspicious TLD investigation
- Time-based DNS analysis
- Dashboard creation
- Threat intelligence enrichment using VirusTotal and AbuseIPDB
